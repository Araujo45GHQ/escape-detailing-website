# Escape Detailing Website

A premium automotive detailing website built with React, TypeScript, and Remix.

## Features

✨ **Premium Dark Aesthetic**
- Deep black backgrounds with electric cyan accents
- Smooth animations and hover effects
- Bold, modern typography (Poppins)

📑 **Multi-Page Design**
- Home: Hero section with service overview
- Services: Detailed service catalog with pricing
- About: Company story, team, and values
- Contact: Contact form and business information

🎨 **Responsive Design**
- Mobile-optimized layouts
- Flexible grid system
- Touch-friendly interactions

## Tech Stack

- **Framework**: Remix 2.0
- **Language**: TypeScript
- **Styling**: CSS3 with CSS Variables
- **Build Tool**: Vite

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Araujo45GHQ/escape-detailing-website.git
cd escape-detailing-website

# Install dependencies
npm install

# Start development server
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
npm start
```
